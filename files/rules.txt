
Welcome to the Sudoku Game !


To solve the game you have to respect the following rules :

- Each row must contains all numbers from 1 to 9 ONLY once
- Each colomn must contains all numbers from 1 to 9 ONLY once
- Each square must contains all numbers from 1 to 9 ONLY once


To solve the Sudoku, enter a number in the grid such as the following: A,6:1

- A is the row
- 6 is the column
- 1 is the number


Enjoy the game !
